# Day 10: Adapter & Decorator Patterns 🔌

## 🎯 Today's Goals
- Adapter Pattern: Convert interface of a class to another interface
- Decorator Pattern: Add behavior dynamically

## 📚 Theory

### Adapter Pattern
Converts one interface to another that clients expect.

### Decorator Pattern
Attaches additional responsibilities to an object dynamically.

---
**"Adapter makes things work after they're designed; Decorator makes things work while they're being designed."**
